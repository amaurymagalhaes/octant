# Hexagon

Golem Foundation's decentralised governance system that will be first rolled out to govern the Wildland project.

## Configuration
ensure that `.env`  file is present. See `.env.template`


## Build
```bash
yarn build
```

## Test
```bash
yarn test
```

## Format and lint
```bash
yarn format
yarn lint
```

## Deployment

### Goerli testnet
```bash
yarn deploy:goerli
yarn verify:goerli
```

### zkSync testnet
```bash
yarn deploy:zksync
```
